# DevConnect Frontend

Frontend of DevConnect, a social networking application built with HTML5, CSS3 and Vanilla JavaScript.

## Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- REST API
- Responsive Layout

## Features

- User authentication
- Feed
- Posts
- Comments
- Likes
- Friends
- User profiles
- Photo gallery
- Search users

## Backend

This project consumes the DevConnect REST API:

https://github.com/VMBacca/devconnect-api
